# Pacific Basin 历史数据集 (2021-2024)

本目录包含 Pacific Basin Shipping 的历史财务和运营数据，用于 BI Dashboard 的数据可视化和分析。

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `pbHistoricalData.ts` | 主要数据文件，包含类型定义和所有数据，可直接在 React 组件中导入使用 |
| `README.md` | 本文件，数据文档说明 |

### 四年累计数据

- **总收入**: $111.3亿
- **累计 EBITDA**: $26.4亿
- **累计净利润**: $17.9亿
- **累计货运量**: 3.32亿吨
- **平均核心船队**: 130艘

## 🔑 核心发现

1. **市场周期性**: 2021-2022年为超级周期，2023-2024回归常态但仍高于历史均值
2. **运营效率**: 2021-2023年平均跑赢市场指数20%以上
3. **成本领先**: OPEX控制在$4,700/天，处于行业领先水平
4. **财务稳健**: 四年平均ROE约27%
5. **船队优化**: 自有比例从85%提升至89%
